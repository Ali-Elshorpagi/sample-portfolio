## Preview
- Sections: hero, worked-with logos, case studies, testimonials, recent work, and a contact form.
- Responsive: collapses nav into a toggle and stacks grids on smaller screens.

## Run Locally
1) Clone or download this folder.
2) Open index.html in your browser (no build step needed).

## Customize
- Update text and links in [index.html](index.html).
- Swap images under [assets/](assets/) and match filenames in the HTML.
- Tweak styling in [styles.css](styles.css); mobile rules live under the `@media (max-width: 768px)` block.
